JTerminal

JTerminal is a Terminal Emulator written in Java.

Although it is still a little rough around the edges, it is based on a very solid design that makes JTerminal very flexible.

Here is what it can do now:
Emulations:
·Data General ® D210 (Hey this is the only one I need)
· If you point me to an easy-to-read reference for a VT-100 or ANSI terminal, I can throw one together pretty fast (or better yet, write the code and send it to me - it's simple... really)
Communications:
· Serial (using your com port) (You need the comm package from javasoft - it's free)
· RMI (over the internet/intranet)
Deployment:
· Application
· Servlet
· Applet
· Webstart Application
· Remote Control Application

I use it as:
A DG Terminal Emulator on my local computer.
A HTML Terminal Emulator (so I can access my miniconputer) when I'm off-site.
A remote-control terminal to automatically reboot my computer if my connection to the Internet is lost.
